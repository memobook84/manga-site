# THE MANGA STORE 開発進捗報告

## 作業日
2026年2月14日〜2月16日

## 完成した機能

### トップページ（index.html）
- 90作品の漫画を6列×15行のグリッドで表示
- 各作品をカラフルな色分けで表示（CSSベース）
- 検索機能（タイトル・著者名で検索可能）
- ヘッダーナビゲーション（New Release・Followへのリンク）

### 作品詳細ページ（detail.html）
- 作品情報の表示（タイトル、著者、出版社、レーベル、ジャンル、発売日、あらすじ）
- 著者リンク（クリックで著者ページへ遷移）
- フォロー機能（ハートアイコンでお気に入り登録、フォロー時に赤く変化）
- 各巻のリスト表示（クリックで巻詳細ページへ）
- 各巻の発売日を自動計算（初回発売日から3ヶ月ごと）

### 巻詳細ページ（volume.html）
- 各巻の詳細情報
- 購入ボタン（楽天・Amazon）
- 作品ページへの戻るリンク

### 著者ページ（author.html）
- 著者の略歴表示
- 代表作品リスト
- その著者の全作品をグリッド表示

### フォローページ（follow.html）
- フォロー中の作品をグリッド表示
- フォロー解除機能
- localStorageでデータ永続化
- 空の状態のメッセージ表示

### 新刊ページ（new-releases.html）
- 最新20作品をリスト形式で表示
- NEW バッジ表示（上位3作品）
- 発売日順にソート
- クリックで作品ページへ遷移

#### カラースキーム
| 用途 | 色 |
|---|---|
| ヘッダー背景 | `#595959`（ダークグレー） |
| ページ背景 | `#FAF8F5`（温かみのあるクリーム） |
| カード背景 | `#FFFFFF` |
| テキスト（主） | `#2B2B2B` |

| フォローハート（済） | `#e74c3c`（赤） |
| 楽天ボタン | `#8B6B5C`（ウォームブラウン） |
| Amazonボタン | `#A68A6F`（トープ） |
| NEWバッジ | `#9B6B6B` |

#### フォント
- **ヘッダー（英語）**: Inter（アルファベットのみ）
- **見出し（英語）**: Crimson Pro（アルファベットのみ）
- **日本語・数字**: Noto Sans JP（全て）
- `@font-face` の `unicode-range` でアルファベットのみInter/Crimson Proを適用

#### ヘッダー
- 背景: ダークグレー（`#595959`）、文字: 白
- コンパクトなパディング（`7px`）
- 下線: `1px solid #444`
- ナビリンク: 大文字表記（NEW RELEASE, FOLLOW）
- 検索窓: `Search manga...`（英語）

#### デザイン方針
- シンプルで自然なデザイン
- 枠線や影を削除（AI臭を排除）
- 余白を活用したクリーンなレイアウト
- ホバーエフェクトで適度なインタラクション
- カードに白背景+薄いシャドウ（トップページのグリッド）

### ファビコン
- 惑星アイコン（`favicon.png`）を全ページに設定

## 技術仕様

### フロントエンド
- HTML5 / CSS3 / JavaScript (Vanilla JS)
- レスポンシブデザイン対応
- Google Fonts使用（Inter, Playfair Display, Crimson Pro, Noto Sans JP）
- CSS変数（カスタムプロパティ）による一元管理

### データ管理
- 共通データベース（manga-data.js）
- localStorage（フォロー機能）

### 画像表示
- CSSベースのプレースホルダー
- 各作品固有の色で表示
- 外部サービスに依存しない（完全オフライン動作可能）

### サーバー
- Python HTTP Server（開発用）
- ポート: 8000
- URL: http://localhost:8000
